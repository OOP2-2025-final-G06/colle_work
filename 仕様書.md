### シフト管理とゲームの複合GUIアプリケーション（Colle-Work）

1. ホーム画面
    - タイトル ( Colle-Work )
    - ユーザー名
    - パスワード
    - ログイン
    - 新規ユーザー登録

2. 新規ユーザー登録
    - ユーザー名
    - パスワード
    - 登録

3. ダッシュボード
    - ** 画面左側 **
        - 月間シフト表 ( 個人のを表示 ) \\優先度低
        - ユーザーリスト
        - 時給変更
        - ゲーム
        - ホームに戻る

    - ** 画面右側 **
        - 今週のシフト表 ( 個人のを表示 \\優先度低)
        - 来週のシフト表 ( 個人のを表示 \\優先度低)
        - シフト登録
        - 暫定月収, 今年の累計収入, ゲーム内トークン

4. シフト表示画面
    - ** 今週のシフト **
    ( 全体のを表示, 変更不可 )

    - ** 来週のシフト **
    ( 全体のを表示, 変更可能 )

5. ユーザーリスト
    - 名前, パスワード
    - ユーザー削除 \\優先度低

6. ゲーム画面
    - 規模順　ショップ → ガチャ → 育成ゲーム
