### シフト管理とゲームの複合GUIアプリケーション（Colle-Work）

1. ホーム画面
    - タイトル ( Colle-Work )
    - ユーザー名
    - パスワード
    - ログイン
    - 新規ユーザー登録

2. 新規ユーザー登録
    - ユーザー名
    - パスワード
    - 登録

3. ダッシュボード
    - ** 画面左側 **
        - 月間シフト表 ( 個人のを表示 ) \\優先度低
        - ユーザーリスト
        - 時給変更
        - ゲーム
        - ホームに戻る

    - ** 画面右側 **
        - 今週のシフト表 ( 個人のを表示 \\優先度低)
        - 来週のシフト表 ( 個人のを表示 \\優先度低)
        - シフト登録
        - 暫定月収, 今年の累計収入, ゲーム内トークン

4. シフト表示画面
    - ** 今週のシフト **
    ( 全体のを表示, 変更不可 )

    - ** 来週のシフト **
    ( 全体のを表示, 変更可能 )

5. ユーザーリスト
    - 名前, パスワード
    - ユーザー削除 \\優先度低

6. ゲーム画面
    - 規模順　ショップ → ガチャ → 育成ゲーム

7. 変数名
    -ユーザー選択
        user_name ユーザー名
        password パスワード

    -シフト管理
        shift_date シフトの日
        shift_hour シフトの時間
        current_date 現在の日付
        selected_week 選択中の週
        salary_per_hour 時給

    -ゲーム
        token 通貨（ゲーム用）

    -ホーム（表示）
        total_salary 給与合計
        monthly_salary 月収
        yearly_salary 年収
        total_shift_hour シフトの合計(時間h)
